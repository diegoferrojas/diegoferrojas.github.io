Purpose
An interface declares a set of operations, including their signatures and parameters, that are use to specify the services offered by a classifier model element (e.g. a class, component or subsystem).

Description
Main Description A model element which defines a set of behaviors (a set of operations) offered by a  classifier   model element (specifically, a class, subsystem or component). A classifier may  realize  one or more interfaces. An interface may be realized by one or more classifiers. Any classifiers which realize the same interfaces may be substituted for one another in the system. Each interface should provide an unique and well-defined set of operations.
